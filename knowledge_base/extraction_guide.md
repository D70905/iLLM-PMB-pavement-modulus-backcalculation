# 知识库提取指南

## 使用说明
对每份 PDF，按下面标注的优先级和定位找到原文内容，然后按 `knowledge_base.jsonl` 的 schema 格式填入。
每条片段三要素：**知识内容（原文摘录或精炼） + 出处（规范号+表号/条款号/页码）+ 标签**。

## 提取格式（knowledge_base.jsonl 每行一条）

```json
{"id":"kb001","content":"知识内容原文或精炼表述","source":"规范号+表号/条款号","tags":["关键词1","关键词2"],"type":"static_and_rag"}
```

`type` 取值：
- `"static"` — 普适不变规则，只进硬约束层（如模量不能为负、土基不高于基层）
- `"rag"` — 情境相关，只进RAG检索层（如特定材料+温度下的典型模量）
- `"static_and_rag"` — 两边都进（既有普适边界又有情境典型值）

## PDF 1：JTG D50《公路沥青路面设计规范》⭐ 最核心

**目的**：提取所有结构层材料模量设计参数 + 温度修正 + 结构组合规则

### 需要提取的内容（按优先级）

#### P0（必须有）
1. **沥青混合料抗压回弹模量设计参数表**
   - 定位：搜索"抗压回弹模量" 或 "设计参数" 或 "沥青混合料"，找到 20°C / 15°C 等不同温度下的模量建议值表
   - 需要摘：面层沥青混合料在不同温度条件下的模量范围（MPa）
   - 标签：`材料:沥青混凝土, 类型:模量范围, 维度:温度, 层位:面层`

2. **无机结合料稳定类基层/底基层设计参数表**
   - 定位：搜索"无机结合料" 或 "水泥稳定" 或 "二灰稳定" 或 "基层"
   - 需要摘：水泥稳定碎石、水泥稳定砂砾、二灰稳定碎石等的回弹模量范围
   - 标签：`材料:水泥稳定碎石, 类型:模量范围, 层位:基层, 结构类型:半刚性`

3. **粒料类材料设计参数表**
   - 定位：搜索"粒料" 或 "级配碎石" 或 "未筛分碎石"
   - 需要摘：级配碎石、未筛分碎石等的回弹模量范围
   - 标签：`材料:级配碎石, 类型:模量范围, 层位:底基层/基层, 结构类型:柔性`

4. **路基回弹模量取值**
   - 定位：搜索"路基回弹模量" 或 "土基"
   - 需要摘：不同土质类型路基的回弹模量参考范围
   - 标签：`材料:土基, 类型:模量范围, 层位:路基`

5. **典型路面结构组合形式**
   - 定位：搜索"结构组合" 或目录中找路面结构组合设计章节
   - 需要摘：沥青路面、半刚性路面等典型结构类型的层间模量梯度关系（面层/基层/底基层/土基的相对刚度）
   - 标签：`类型:结构组合, 类型:层间关系`

#### P1（尽量有）
6. **沥青层模量温度修正系数/公式**
   - 定位：搜索"温度" + "修正" 或 "温度调整系数"
   - 需要摘：沥青层模量随温度变化的修正方法
   - 标签：`类型:温度修正, 材料:沥青混凝土`
